# Handoff: Avrrora — сайт салону краси

## Overview
Багатосторінковий маркетинговий сайт салону краси «Avrrora» (Київ). 7 сторінок: Головна, Про нас, Роботи (портфоліо), Послуги (ціни + калькулятор), FAQ, Контакти, 404. Мова — українська. Стиль — «Apple-like»: багато повітря, скляна навігація, стриманість, плавні ненав'язливі анімації. Палітра: синій, фіолетовий, білий, трохи чорного.

## About the Design Files
Файли в цьому пакеті — **дизайн-референси, створені в HTML** (`*.dc.html`). Це прототипи, що показують задуманий вигляд і поведінку, а НЕ production-код для прямого копіювання. Завдання: **відтворити ці дизайни в цільовому кодовому середовищі** (React, Vue, Next.js тощо), використовуючи його усталені патерни. Якщо середовища ще немає — обрати доречний фреймворк (для такого сайту добре підійде Next.js/Astro зі статичною генерацією) і реалізувати там. Розмітка в файлах — усередині тега `<x-dc>`, стилі інлайнові; `support.js` — рантайм прототипа, у продакшн не переноситься. Атрибути `style-hover` = стилі `:hover`; `sc-for`/`sc-if` = цикли/умовний рендер; клас `Component` внизу файлу = стан і обробники (аналог React-компонента).

## Fidelity
**High-fidelity.** Кольори, типографіка, відступи, радіуси, тексти та анімації — фінальні. Відтворювати піксель-у-піксель.

- Reveal при скролі: елементи з `data-reveal` стартують `opacity:0; translateY(26px)` → з'являються через IntersectionObserver (threshold 0.12), transition `0.9s cubic-bezier(.22,.61,.21,1)`
- Hover кнопок: `translateY(-3px)`, 0.35s
- Hero: в'їзд контенту 1.1s тим самим beзьє; «сяйво» пульсує (`opacity .55↔.9`, 7s)
- Marquee: стрічка 32s linear infinite, `translateX(-50%)` (контент задубльовано)

## Interactions & Behavior
- Навігація — звичайні лінки між сторінками; активний пункт підсвічено.
- Scroll-reveal на всіх сторінках (описано в токенах); елементи, що вже у в'юпорті при завантаженні, не ховаються.
- Роботи: фільтр (стан `filter`), лайтбокс (стан `lightbox`), слайдер (стан `ba` 4–96, `baDrag`; Pointer Events + setPointerCapture; `touch-action:none; user-select:none`).
- Послуги: калькулятор — стани `svc`, `len`; результат перераховується миттєво.
- FAQ: стан `open` (індекс або −1).
- `html{scroll-behavior:smooth}` всюди; `text-wrap:pretty` на абзацах.
- Форм і бекенду немає навмисно — запис через дзвінок/Instagram; бекенд-байти на дзвінок клієнт додасть пізніше сам.

## State Management
Кожна сторінка самодостатня. Стани: Роботи — `{filter, ba, baDrag, lightbox}`; Послуги — `{svc, len}`; FAQ — `{open}`. Дані (прайс, роботи, FAQ) — статичні масиви, можна винести в CMS/JSON.
