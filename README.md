# Avrrora — сайт салону краси

Багатосторінковий сайт салону краси «Avrrora» (Київ). Статичний сайт (HTML/CSS/JS),
без збірки — деплоїться на Vercel як є з кореня репозиторію.

## Сторінки

| Файл | Сторінка |
|------|----------|
| `index.html` | Головна |
| `about.html` | Про нас |
| `works.html` | Роботи (портфоліо: фільтр, «до/після», лайтбокс) |
| `services.html` | Послуги та ціни (+ калькулятор) |
| `faq.html` | FAQ (акордеон) |
| `contacts.html` | Контакти (+ мапа) |
| `404.html` | Сторінка не знайдена |

## Структура

```
/                 корінь = сайт, який роздає Vercel
  *.html          сторінки
  assets/site.js  спільна поведінка (hover-стилі, поява при скролі)
  images/         фото
  design/         оригінальний дизайн-хендофф (*.dc.html) — референс, не деплоїться
```

Інтерактив (калькулятор, акордеон, галерея, повзунок «до/після», лайтбокс) —
чистий ванільний JS усередині відповідних сторінок. Зовнішніх залежностей немає.

## Локальний перегляд

Будь-який статичний сервер із кореня, напр.:

```bash
python -m http.server 8099
# → http://localhost:8099
```

## Деплой

Vercel підключено до цього репозиторію: пуш у `main` → автоматичний деплой.
Framework preset — «Other», build command не потрібен, output — корінь.

## Дизайн

Стиль «Apple-like»: багато повітря, скляна навігація, плавні анімації.
Палітра: синій `#2f4fd8`, фіолетовий `#7a3ff2`, фон `#f7f7fb`, темні секції `#0b0b16`.
Шрифти: Unbounded (заголовки) + Manrope (текст). Повний опис токенів — у `design/README.md`.
